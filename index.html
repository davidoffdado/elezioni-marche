Profilo GitHub

Nel mio profilo GitHub sono presenti i dati e il funzionamento degli scraper, in modo da poter replicare le analisi e usare gli script come punti di partenza per eventuali futuri scraper.

Scarica i dati

  • comuni_map.csv - comuni e rispettivi ID in formato .csv;
  • comuni_map.json - comuni e rispettivi ID in formato .json;
  • elezioni-marche-affluenza.xlsx - dati sull’affluenza, livello provinciale e comunale;
  • flourish_input_all.csv – percentuali preferenze, vincitore e scarto a livello comunale;
  • risultati_comuni.csv - risultati dei candidati per ogni comune;
  • risultati_percentuali_tutti_candidati.csv - percentuali ottenute dai candidati in ogni comune;
  • risultati_provinciali.csv – percentuali ottenute dai candidati in ogni provincia.

Raccolta dei dati in tempo reale

Gli scraper in Python raccolgono automaticamente i dati delle elezioni dai siti ufficiali (come quello della Regione Marche). Il processo è il seguente:

  1. Richiesta HTTP – lo script si collega al sito web e scarica le pagine con i risultati.
  2. Parsing del contenuto – utilizza librerie come BeautifulSoup o pandas per estrarre tabelle, numeri e link.
  3. Pulizia e normalizzazione – i dati grezzi vengono trasformati in formati leggibili (CSV, JSON, Excel), con nomi di colonne coerenti e senza duplicati.
  4. Salvataggio – i file finali vengono salvati nella repository e usati sia per i grafici interattivi su Flourish sia per il download diretto dal sito.

In questo modo, ogni aggiornamento dei dati può essere replicato velocemente lanciando lo scraper, senza dover copiare manualmente le informazioni.
